- 프로그래밍언어

1. basic : 기본 입출력, 자료형, 연산자 등
2. condition : if, if else, switch 등
3. lteration : for, while, do ~ while 등
4. function : 함수란, 재귀 함수, call by vaule / reference 등
5. basic object : number, string, array 등

2. 비선형 자료구조 : 그래프, 힙, 트리, 트라이
